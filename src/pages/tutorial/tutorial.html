Welcome to CHL Reporting App

The Champions Reporting App is a data gathering and report generator tool for reps and their accounts.

How to Use It?

Simply find the Account you want to build a report for and select it to start building a report.

Gather the Project Data

For each project in the account, add photos and project updates / comments.

Click Build, Then Email It!

Once all projects have data input, click the Build Report button to get a preview. Then email the report right from your app.
